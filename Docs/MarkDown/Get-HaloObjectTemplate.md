# Get-HaloObjectTemplate

## SYNOPSIS

Gets an object template for the Halo API.

## SYNTAX

```powershell
Get-HaloObjectTemplate [-Type] <String> [-NullVariables] [<CommonParameters>]
```

## DESCRIPTION

Provides an example object for use in provisioning items in the Halo API
